# app-deployment
This repository contains Packer configurations integrated with Ansible to automate the setup and packaging of datamigrator application. It creates VM images for AWS, Azure, and GCP, pre-configured with the application and its dependencies.
For detailed documentation, please refer to the `docs` directory.

## Directory Structure
- `docs/`: Contains all additional README files and detailed documentation.
- Other files and directories: Deployment-related configurations and resources.

## Notes
- Ensure to review the documentation in the `docs` directory for specific deployment instructions and guidelines.
