# NDM API Automation Testing Tool

API Automation Testing Tool to perform regression and smoke tests.

## Features
- You can create yml file in scenarios durecorty and use that YML file to execute testcase

## Prerequisites
- Go (version >= 1.16)


## Installation
1. Clone the repository
2. Run the following command to install all required dependencies: ```go mod tidy ```
3. Run testcases ```go test -v -count=1 ./...```
